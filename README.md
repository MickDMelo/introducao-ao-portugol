# Introdução ao Portugol

![Capa](capa.png)

## Sobre o Projeto

Este eBook foi desenvolvido como Projeto de Extensão do curso de Análise e Desenvolvimento de Sistemas.

O objetivo do material é oferecer uma introdução acessível à lógica de programação utilizando Portugol, permitindo que estudantes e iniciantes desenvolvam competências fundamentais para o aprendizado de linguagens profissionais.

## Conteúdo

- Introdução à Programação
- Introdução ao Portugol
- Variáveis e Tipos de Dados
- Operadores
- Estruturas Condicionais
- Estruturas de Repetição
- Vetores e Matrizes
- Procedimentos e Funções
- Algoritmos Clássicos
- Boas Práticas e Clean Code
